# Lakshay Goyal - Portfolio Website

A modern, high-performance personal portfolio website built with Next.js 15, TailwindCSS v4, and Framer Motion.

## Features

- **Modern Tech Stack**: Next.js 15 (App Router), React 19, TailwindCSS v4, TypeScript.
- **Animations**: Smooth page transitions and scroll animations using Framer Motion.
- **Theming**: Dark/Light mode support with neon highlights.
- **Responsive Design**: Fully responsive layout for all devices.
- **Sections**:
  - **Hero**: Animated introduction with neon effects.
  - **About**: Bio and education timeline.
  - **Skills**: Interactive skill progress bars.
  - **Projects**: Dynamic project cards with hover effects.
  - **Experience**: Vertical timeline of professional experience.
  - **Contact**: Working contact form simulation.

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/lakshaygoyal/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open http://localhost:3000** to view the site.

## Customization

- **Content**: Update the JSON files in `src/data/` to change projects, skills, and experience.
- **Colors**: Modify the CSS variables in `src/app/globals.css` to change the theme colors.

## Deployment

This project is optimized for deployment on Vercel.

1. Push your code to a GitHub repository.
2. Import the repository in Vercel.
3. Click **Deploy**.
